# Development

- Install node (>= 6.0.0)

- Install yarn (>= 0.21.3)

```bash
brew update
brew install yarn
yarn --version
```

- Start spring application in debug mode

- Start react application development mode

```bash
yarn start
```

- Open `http://localhost:3000` in browser
